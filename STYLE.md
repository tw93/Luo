# Luo 落文字体特征抽象与批量改字规范

这份文档用于给其他 AI 或后续批量改字流程学习落文风格。它描述的是气质、结构判断和批量处理边界，不是某个字的临摹稿。

## Summary

- 落文是一套“清骨、端正、可打印”的现代中文字体，不是书法复刻、手写字、石刻字或古风装饰字。
- 风格比例：魏晋小楷式清骨 50% / 当代舒展楷意印刷字的开阔骨架 40% / 现代字体工程稳定性 10%。
- 核心目标：字面饱满、骨架稳定、横轻竖重、收笔干净，长文排版安静，小字号打印清楚。
- 其中“当代舒展楷意印刷字”指一种现代字库化的楷意风格：字面开阔、结构舒展、笔画有书写方向但轮廓干净，撇捺有速度，转折有轻微骨节，整体亲和、清爽、适合屏幕与印刷阅读。
- 西湖石刻只提供局部提醒：短、涩、利、收得住；不得引入破损、噪点、纹理、行书连带。

## 当代舒展楷意印刷字特征抽象

这类字体不是传统楷书复刻，也不是手写字，而是一种经过现代字库工程整理后的楷意字体。它保留楷书横、竖、撇、捺、钩、折的方向感，但轮廓更干净，结构更适合屏幕和印刷阅读。

字面偏大，整体舒展，不做窄体；横向空间打开，左右部件不拥挤。字形有明显骨架感，但不僵硬，笔画之间有呼吸感。它的气质不是古典碑帖的厚重，也不是手写文楷的松软，而是清爽、现代、亲和、可读。

横画通常略轻，带轻微上扬，不是完全水平的机械线。竖画承担主要骨架，略重于横画，但不会形成黑硬的竖柱。撇捺有速度感，起笔和收笔干净，舒展但不过分夸张。点画短促、有方向，不圆、不糊、不水滴化。钩画偏短，收束明确，不拖尾。

转折处有轻微骨节，但整体被现代化处理过，不会出现明显墨迹、飞白、破损或书法临写痕迹。端点多为柔和切角，有轻微顿挫，避免圆熟，也避免刀刻感。

这类字体的关键是“楷意被字体化”：看起来有书写来源，但已经不是手写；有传统骨法，但服务于现代排版。它适合标题、说明文字和中短篇阅读，气质温和、舒展、干净，有文化感但不古风化。

## Style DNA

- 骨架：端正舒展，不做窄体；重心稳，字内空间均匀；简单字不虚，复杂字不糊。
- 灰度：整体黑度中等偏清，竖画承担骨架，横画略轻，撇捺有速度但不飞。
- 横竖：横画略有 1 到 2 度上扬，长横稳，短横有顿挫；竖画稳直，可有极轻微自然弧度，不机械。
- 点画：短促、有方向、小楷楔形感；不圆点、不水滴、不斜杠。氵、讠、心、雨字头等点画只做角度归一和轻微形态整理。
- 钩画：短、准、清、利；钩根不肿，钩尾不甩，力量来自方向和收束，不来自加粗或拉长。
- 转折：有骨节但不硬块化；外侧略柔，内侧保留锐度；转折处不能堆黑。
- 端点：软切角，有轻微顿挫和涩味；不尖刺、不圆熟、不刀切。端点要有统一气质，但不能有统一形状；横端、竖底、撇捺端、点端、钩端分开判断。
- 撇捺：撇尾细而不尖，捺脚不圆胖；简单字可舒展，复杂字要收住。
- 复杂字：优先打开内部白，副笔通常减轻 3% 到 5%，雨字头、艹头、鹿、鬲、赢、魔等密集部件可局部放到 5% 到 8%；主笔保留力度，不能靠堆黑显得有力。
- 排版：CJK 字距保持 1em 节奏，标点可略窄；长文灰度稳定，横排竖排都成立。
- 英文和数字搭配：优先选择 humanist、轻调制、不过度几何的西文字体；小字号优先基线和 x-height 稳定，纸面气质由中文 Luo 承担；避免过硬的科技感无衬线，也避免太古典的高对比衬线。
- 定位边界：Luo 是独立的纸面阅读中文字体；外部产品或项目只能作为设计语境和鸣谢来源，不能写成 Luo 的用途边界。
- Latin 边界：`Luo Regular` 保持 CJK-only，不内置 ASCII；英文和数字走推荐 fallback。未来如果做内置 Latin，应作为可选独立版本，不覆盖主版。

## Batch Editing Rules

- 先判断 glyph 类型，再处理：点类、钩类、转折类、多横类、框形类、复杂密集类、撇捺舒展类要分通道处理。
- v0.3 只做 starter 子集收口：官网、README、纸面试读、内部 A4 打印样张、核心锚字和 GB2312 校准页体验。不要在 v0.3 final 里启动大范围扩字。
- v0.4 扩字先从 GB2312 一级常用字开始，再进入完整 GB2312 实验构建；扩字按部件和结构通道批量扩散，不按单字硬补。
- 禁止一刀切固定比例。对输入差异大的属性使用自适应 lerp，例如点画长短轴、复杂字内部收放、局部 bolden 限制。
- 点类 contour 的 bolden 只能限制横向过度膨胀，不能压掉纵向厚度，避免 `代黑点述游清流` 这类斜点/短撇变细。
- SOFT 点画通道语义固定为“角度归一，轻碰形态”，不能继续压短成长斜杠或圆点。
- 钩画只做最终尾部收束，不恢复全局长钩，不增加钩重。
- 转折只做 targeted refinement，不开全局骨节 pass，避免误伤点画和低点数 contour。
- 复杂字只开内部白，不动整体字宽，不改 CJK advance。
- 不通过整体加粗、整体收窄、整体放大字面来解决局部问题。
- 凡是让字变得更亲和、更圆润、更手写、更舒展的处理，都要二次审查；凡是让字更清、更稳、更有骨节但不变粗的处理，优先保留。
- `proof/gb2312.html` 只作为覆盖检查页，公开状态保持 `已覆盖` / `待补字` 两类；结构优化分类只用于工程回归，不作为页面判断标准。

## AI Prompt Template

正向提示：

```text
请将这些中文 glyph 调整为 Luo 落文风格：端正舒展、清骨、有打印稳定性；横轻竖重，点画短促有方向，钩画短准收住，转折有骨节但不堆黑，端点软切带轻微涩感，复杂字内部透气，整体安静耐读。

气质参考魏晋小楷的清瘦骨法，以及一种当代舒展楷意印刷字的开阔结构：字面偏大，横向舒展，轮廓干净，撇捺有速度，笔画有书写方向但不过度手写化。最终必须保持现代字体轮廓稳定、干净、可打印。
```

反向提示：

```text
不要做成行书、草写、手写文楷、碑帖复刻、石刻复刻、景区导视字、古风装饰字。不要加入破损、噪点、飞白、墨迹、纹理。不要让字变圆、变胖、变软、变窄、变粗糙。不要拉长钩尾，不要把点画做成圆点、水滴或斜杠。不要复制任何现成商业字库的具体轮廓，只学习“清骨、舒展、现代楷意、可读”的抽象气质。
```

## Anchor Checks

- 核心锚字：`落文字书心清骨风纸印国回月雨霜藏魔赢道远家亭序集章兰永天玄黄`
- 基准气质：`文字书心用爱清端骨筋开源落纸风永国家设计回馈社会旅行妙言终章印排版兰亭集序`
- 斜点/短撇：`代黑点述游清流`
- 点画：`清润源落读说语社视意念终寒露霜霞`
- 钩画：`字书亭序家设计排版印旅妙馈成式透远道遇`
- 框与转折：`国回图园日目用月风骨短版排印`
- 复杂字：`藏霞霜露馈赢耀魔读题额锦继续源族章集端筋群贤禊觞湍幽怀`
- 心部：`心必思意念想感悲惠慨悟志怀愿恩愁惊慎忠忽怠忍`

## Acceptance Criteria

- 大字看：`落文` 要清骨舒展，不能松软、圆胖或古风。
- 标题看：`以文为名，以字落纸` 要有书卷气，但不手写化。
- 小样看：`代黑点述游清流` 的斜点/短撇不能偏细、不能斜杠化。
- 长文看：《兰亭集序》连续阅读灰度安静，重复字不跳，复杂字不堵。
- 打印看：10pt、12pt、14pt、16pt、24pt、48pt 黑白输出都清楚。
- 回归底线：仍然是落文，不是任何现成字库复刻，不是 LXGW WenKai 松软手写，也不是石刻字。
